## See it yourself


### Clone the package

```
git clone https://github.com/Naturalclar/react-native-sticky-header-example.git
cd react-native-sticky-header-example
```

### Install Dependencies

```
yarn install
```

### Run on ios

```
yarn ios
```

### Run on Android

```
yarn android
```
